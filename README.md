# Entrevista-CRUD-NodeJS
Guia rapida para configuracion y ejecucion del proyecto.

## Prerrequisitos

**Node.js** Versión `22.11.0` o superior.

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/FranciscoJRuiz00/Entrevista-CRUD-NodeJS.git
    ```

2.  **Entrar a la carpeta del proyecto:**
    ```bash
    cd crud-microservice
    ```

3.  **Instalar dependencias:**
    ```bash
    npm install
    ```

4.  **Configurar el entorno:**
    Crea un archivo `.env` en la raíz del proyecto con este formato:
    ```env
    DB_HOST=
    DB_PORT=
    DB_USER=
    DB_PASSWORD=
    DB_NAME=

5.  **Ejecutar el servidor de desarrollo:**
    ```bash
    npm run dev
    ```

Una vez ejecutado, abre tu navegador en `http://localhost:3000/`.
